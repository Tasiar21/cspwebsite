1.1.9 Project

Welcome to the Sushi Factory!


This is an interactive code which takes the user through a lesson of how to make either a California roll or a Salmon roll. The user will be prompted to type each ingredient according to the order given, they will also be able to roll and cut the sushi into how many pieces they would like.


1.2.5 Project

Reindeer Run


This program uses user input through onkey functions to navigate past the moving colored gifts to reach the end. The user has 3 lives and 30 secoonds to make it past the moving obstacles. The obstacles (gifts) are in a constant loop to move at a constant speed up and down the screen. If the user fails to complete the game in 3 lives, the screen will display "You Died"
